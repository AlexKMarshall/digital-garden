## Key bindings 

| Binding | Description | Windows Equiv|
|-------|------|------|
| alt + arrow key | Move cursor one word | ctrl + arrow key |
| windows + \[c, v, a\] | copy, paste, select all etc | ctrl +  \[c, v, a\] |

## Key mappings from windows keyboard

| Key you want | Where it is on windows layout |
|-----|-----|
| \| | shift + \# |
| \# | shift + 3 |
| \` | \\ |
| \\ | ~  (shift + \#) |
| ~ | \| (shift + \\) |
| @ | shift + 2 |
| command | windows key|
